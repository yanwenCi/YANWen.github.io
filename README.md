# Yan Wen Personal Page

Static academic personal page generated from `CV_YAN WEN_ML.docx`.

## Recommended Platform

Use GitHub Pages. This site is plain HTML/CSS, so it can be published directly from a repository without a build step.

## Publish With GitHub Pages

1. Create a public GitHub repository, for example `yanwen.github.io`.
2. Copy the contents of this `personal-page` folder into the repository root.
3. Commit and push.
4. In GitHub, open `Settings` -> `Pages`.
5. Choose deployment from the `main` branch and root folder.

## Files

- `index.html`: page content and structure.
- `styles.css`: responsive styling.
- `assets/prostate-mri-hero.png`: generated abstract prostate MRI hero image.

The page includes selected original repositories from `https://github.com/yanwenCi` and omits the phone number from the CV for privacy.
